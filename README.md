# brain-tumor-segmentation
An automated system using U-Netarchitecture to detect and segment brain tumors from MRI scans, which improves diagnostic accuracy and efficiency.

## Quick Start
1. **Clone the repository:**
   `git clone https://github.com/cheez143/brain-tumor-segmentation.git`
2. **Install dependencies:**
   `pip install -r requirements.txt`
3. **Prepare Data:**
   See [Data Documentation](data/README.md) for instructions on where to download and how to organize the dataset.
4. **Train:**
   `python train.py --epochs 50`
